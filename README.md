# privacy-policy
4Part-Memo プライバシーポリシー
最終更新日：2026年2月21日
本プライバシーポリシーは、4Part-Memo（以下「開発者」）が提供するスマートフォン用アプリケーション「4Part-Memo」（以下「本アプリ」）における、ユーザー情報の取り扱いおよびデータ削除に関する指針を説明するものです。
1. データの収集と利用目的
本アプリは、以下のデータを収集し、それぞれの目的で利用します。
アカウント情報（メールアドレス、ニックネーム）: ユーザー認証およびアプリ内での個人識別のため。
アプリのコンテンツ（メモ内容）: メモの保存、管理、およびパートナーとのリアルタイム同期機能を提供するため。
デバイスID: 広告配信（Google AdMob）およびアプリの動作解析のため。
2. アカウントおよびデータの削除リクエスト
本アプリは、ユーザーが自身のデータを完全にコントロールできる権利を尊重し、以下の通り削除手段を提供します。
削除の手順: ユーザーは本アプリ内の「設定」メニューにある「アカウント削除」ボタンから、いつでも即座にアカウントの削除とデータの抹消をリクエストできます。
アプリを利用できない場合: アプリのアンインストール等によりアプリ内からの操作が困難な場合は、開発者（4partmemo@gmail.com）へメールでご連絡いただくことで、手動での削除リクエストも受け付けています。
3. 削除されるデータと保持されるデータ、および期間
ユーザーから削除リクエスト（アカウント削除）があった場合、以下の通り処理を行います。
削除されるデータの種類:
ユーザープロファイル（ニックネーム、メールアドレス）
個人用のメモ（「Buy」「Task」「Idea」「Wish」に含まれる全テキストデータ）
認証用アカウント情報
保持されるデータの種類:
共有メモ: 連携ID（familyId）を使用して他のユーザーと共有しているメモについては、共有相手の利便性を維持するため、アカウント削除後も共有グループ内に保持されます。
データの保持期間:
個人データ: アカウント削除リクエストが完了した時点で、サーバー上から即座に抹消されます。バックアップデータについても、システム上から最大30日以内に完全に消去されます。
共有データ: 共有グループ内のすべてのメンバーが削除されるか、グループが完全に解散されるまで保持されます。
4. 外部サイトへのリンクについて
本アプリには、外部サイト（楽天、Amazon等）へのリンクが含まれる場合があります。移動先のサイトにおける個人情報の取り扱いについては、本アプリの開発者は一切の責任を負いかねます。各リンク先サイトのプライバシーポリシーをご確認ください。
5. セキュリティと暗号化
すべてのデータは転送時に暗号化（HTTPS/TLS）され、安全にサーバーへ送受信されます。
